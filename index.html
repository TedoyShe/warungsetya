About us

Website statis yang telah saya buat adalah untuk Storage Resource Management yang memanfaatkan Google Spreadsheets API sebagai basis data. Melalui integrasi dengan API tersebut, website ini memungkinkan pengguna untuk melacak dan mengelola sumber daya penyimpanan dengan mudah dan efisien. Dengan pendekatan statis, website ini menawarkan keterjangkauan serta kemudahan dalam pengelolaan data penyimpanan tanpa memerlukan server backend yang kompleks.
